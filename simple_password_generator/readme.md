## Features

1. Generate strong passwords
2. Choose the length
3. Include
   1. UPPERCASE
   2. lowercase
   3. digits
   4. punctuations

## Setup to run
1. Clone the repo : ```git clone my-python/simple_password_generator```
2. Navigate into the folder
3. Run the program using ```password_generator.py```
